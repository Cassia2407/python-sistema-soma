# Sistema de Soma e Comparação em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## Funcionalidades
- Recebe dois números do usuário
- Calcula a soma
- Identifica o maior número
- Exibe os resultados no terminal

## Tecnologias
- Python

## Objetivo
Praticar lógica de programação e estruturas condicionais.

## Como executar
1. Baixe o arquivo
2. Execute no terminal com:
   python sistema.py
